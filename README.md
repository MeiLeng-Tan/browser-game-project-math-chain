MATH CHAIN
A browser-based number puzzle game where the player combines numbers using the arithmetic operations to reach a target number. 

Game Features:
- Randomly generates numbers and a target number each game.
- Supports addition, subtracton, multiplication and division.
- Player can toggle their selections.
- Tracks player moves and time. 
- Undo features for moves
- Dynamic history board showing moves and operations.
- Win message with a summary of total time and moves taken. 

How to Play:
1. Select the first number on the board. You can toggle the selection by clicking the same number again. 
2. Select and arithmetic operator (+, -, *, /). You can also toggle the operator selection by clicking it again. 
3. Select the second number on the board. 
- For subtraction, only numbers that result in a positive value can be selected. 
- For division, numbers that would result in a remainder cannot be selected. 
4. Click = to solve the equation. 
5. The first selected number on the board is updated with the result and the second button disappears. 
6. Repeat steps 1-5 until you reach the target number.
7. Historical moves are displayed in the history box. 
8. Click the Undo button to revert your last move. 
9. After winning, click Start New Game to reset the board and begin a new game.
